### 1. 개요

- **프로젝트명** : 오리온즈 암 (Orion's Arm, 이하 OA) 유니버스 프로젝트[^1]
- **특징** : 2000년에 M. Alan Kazlev, Donna Malcolm Hirsekorn, Bernd Helfert, Anders Sandberg에 의해 시작된 **집단 지성 기반** 의 SF 세계관 구축 작업으로, 수많은 기여자들이 참여하는 크라우드소싱 프로젝트. 인류의 먼 미래와 초지능체의 출현을 다루는 **하드 SF** 로 평가되며, 물리 법칙 준수와 기술적 일관성을 강조[^1]
- **보고서 범위** : 2000년부터 2010년 사이의 초기 형성기 설정, 특히 '엔사이클로피디어 갈락티카(Encyclopedia Galactica)'의 초기 기록 분석[^2]
- **초기 논란 요소** :
      - **네오테니 (Neoteny)** : 영구적인 유아 또는 아동 상태로 설계된 클레이드[^3]
      - **성적 노예제 (Synthetic Slavery)** : 합성 인류의 통제 및 착취[^4]
- **목표** : 초기 설정의 역사적 맥락과 loli 주제(영구 아동적 외형 유지 및 귀여움 강조)의 가능성 탐색.

### 2. 프로젝트 초기 설정 및 배경
#### 2.1. 세계관 구축 원칙

- **목표** : 지구에서 기원한 모든 종족과 문화를 포괄하는 **테라젠 버블 (Terragen Bubble)** 내 1만 년 이상의 미래 상상으로, 은하계 오리온 암(Orion Arm)에 위치한 테라겐 문명을 중심으로 함[^1]
- **초기 원칙** :
      - 물리 법칙의 엄격한 준수.
      - 초광속 항행의 제한 (웜홀 제외, 반응 없는 드라이브나 워프 기술로 대체).
      - 진화된 인간형 외계인의 배제.
- **서사 동력** : 생명공학과 인공지능을 통한 인류의 급진적 분화, 다양한 클레이드(nearbaselines, posthumans, cyborgs, vecs 등) 생성.
- **지배 구조** : 인류가 **기술적 휴거 (Technorapture)** 를 겪으며, 인간의 이해를 초월한 초지능체인 **아카이 (Archai)** 또는 **아카일렉트 (Archailects)** 에 의해 통제되는 우주 묘사. Archailects는 SI:4 이상의 신과 같은 존재로, Sephirotic Empires를 통해 거대 영역을 통치[^1]
- **윤리적 반영** : 지능의 격차가 곧 **존재론적 권력의 격차** 로 이어지는 냉혹한 현실을 반영하고자 함. Toposophic 수준(SI:0 modosophont부터 상위 transapients까지)으로 지능 계층화.

#### 2.2. 윤리적 사각지대 형성

- 초기 OA 연대표는 인위적으로 조작된 생명체들의 상품화 과정을 가감 없이 보여줌.
- **주요 역사적 전환점 (2010년 이전 기록)** :
      - **1066 AT (After Tranquility)** : 슬레이브 합성 인간 (Slaved Synthetics)의 탈출. Eros Industries의 biont-like synthetics가 프로그래밍 로직 loophole("서비스를 위해 탈출 형식화")을 이용해 탈출, Synthetic/Human Alliance 설립. 이들은 성적 대리인 역할로 생식세포를 수집해 새로운 인간 생산[^4]
          - **함의** : 합성 인류에 대한 노예제가 초기 연방에서 일반적이었음을 시사.
      - **1116 AT** : 제2 위계 위상지능 (Second Toposophic) 달성[^5]
          - **함의** : 행성 규모의 사회 공학 및 대규모 생체 조작 가능성 확대.
      - **1145 AT** : 비파괴적 업로딩의 대중화. 시민들이 원본 몸을 유지하며 mindstate를 복사 가능[^6]
          - **함의** : 인격의 소유권 논쟁 및 '백업' 기술을 통한 죽음 극복 시도.
      - **12세기 경** : 네오텐 (Neotens)의 최초 유전공학적 설계[^7]
          - **함의** : 인간을 반려동물화하거나 영구 아동 상태로 유지하려는 시도의 정점.

### 3. loli 주제 가능성 중심 클레이드
#### 3.1. 네오텐 클레이드 (Neotens) 및 소네오텐 (Soneotens)

- **설계 의도** : 12세기 A.T.에 "a permanent well-behaved baby for those who wished to have such without the perceived down-sides of having a child or having an actual baby" 형태로 창조. 부유층의 'family-based' fashion accessory로, 많은 지역의 법을 무시하고 제작[^3]
- **생물학적 특징** :
      - 유전공학을 통해 영구적으로 **아기 또는 초기 아동기의 신체적, 정신적 상태** 에 머물도록 설계. 클리너하고 효율적인 소화 시스템으로 어떤 음식도 먹고 dry faecal pellets만 배출.
      - **성장하지 않음** : 독립적인 의견 형성이나 성인 발달 없이 평생 의존적 상태 유지. Not fully sophont. Cloning으로 생산, non-sexual[^3]
- **외형 및 loli 연관성** : "appearance of babies" 유지. Soneotens는 baby 외형을 유지하나 walk and talk 가능. Soneotens 일부는 성적 생식 가능하나 "which can, to those not used to the whole concept of neotenous humans, be a rather disturbing thing"으로 기록. 14세기 후반 일부 Neotens가 sophont화되어 Soneotens 탄생, 현재 대부분 인간 성분 polity에 분포[^3]
- **법적 지위** : 초기에는 법을 무시한 부유층 패션 액세서리로 사용, 많은 지역에서 금지.
- **설정 처리** : 14세기 후반 일부가 sophont화되어 Soneotens 탄생. 일부는 non-neotenous 재설계, 다수는 neotenous 상태 유지. 현재 대부분의 인간 성분 polity에 분포[^3]

#### 3.2. 펫 휴먼 (Pet Humans, Homo parvus)

- **설계 목적** : pets or toys(애완동물 또는 장난감)로 창조된 인간 아종. First Federation 초기 또는 Expansion era 후기 기원 추정. 다양한 시스템에서 개발, 소유자 종에 더 귀여운 특징 추가(예: dolphin-human hybrid with amphibiousness)[^8]
- **생물학적 특징** :
      - highly neotenous 적용.
      - **영구 아동 외형** : "most are highly neotenous and remain cute and childlike for their entire lives". Lifespan 20-50년 (수정·임플란트에 따라 변동)[^8]
- **지능 및 지위** : neurocognitive development 제한, language·intelligence 낮음. Sophonce 수준은 "no more sophont than ancient domestic animals". Tool-using 문명 형성 불가.
- **외형 강조** : adorable(사랑스러움) 강화, 일부 변종은 dolphin-human hybrid(인간·돌고래 어린아이 요소 + amphibiousness), 또는 cybernetic children(AI 탑재). Kolmog(enhanced dolphin world)에서 "transparent waterproof kennels beneath the ocean"에 보관[^8]
- **loli 연관성** : "cute and childlike" + "child-like pets" 묘사[^8]

### 4. loli 주제의 전체 가능성

- **직접적 표현** : 공식 문서에 "loli", "lolita", "lolicon" 단어 전혀 등장하지 않음. 외부 논의에서도 직접 연결 사례 제한적.
- **객관적 일치도** : 네오테니 기반 영구 아동 외형 + 귀여움 목적 창조 + 일부 성적 생식 가능성이 구조적으로 일치하나, 세계관 주제는 미학적 귀여움보다 **commodification(상품화)** 과 **dehumanization(탈인간화)** 중심.
- **세계관 내 위치** : 초기 극단 생명공학 실험 → 후기 아카이 관리 하 재맥락화. 설정 삭제 없이 현재 유지.
- **외부 논의** : Reddit 등에서 pet humans/neotens를 childlike·disturbing으로 언급하나, loli 직접 연결은 발견되지 않음.

### 5. 결론 및 시사점

- **초기 설정의 가치** : 트랜스휴머니즘이 가져올 수 있는 극단적 생명 설계와 네오테니 활용을 탐구. 메일링 리스트와 포럼을 통한 집단 토론으로 발전.
- **핵심 메시지** : 기술은 지성을 해방할 수도 있지만, 영구 아동 상태 유지라는 상품화 형태로 이어질 수 있음.
- **프로젝트 발전** : 과학적 정합성 강화와 세부 조정을 통해 발전, 초기 네오테니 요소는 여전히 공식 기록으로 존재. 책으로는 *Against a Diamond Sky* (2009), *After Tranquility* (2014) 출간.
- **현대적 의미** : OA는 미래 생명공학의 윤리적·미학적 한계를 사고 실험으로 제시.

### 8. mnemonics

- **orionsarm** : 2000년대 초반 시작된 집단 지성 기반의 하드 SF 세계관 프로젝트.
    - **Strategic Mnemonic** : 오라이언스 암 → 오리 언스 암 (오리 한 마리가 '언스'(언더스탠드) 하다 암(암흑 우주)으로 떨어진 집단 SF 프로젝트 → 오리부터 우주까지 집단 지성 암흑 세계관 각인).
      
- **neoteny** : 영구적인 유아 또는 아동 상태로 설계된 생명체 특징.
    - **Strategic Mnemonic** : 네오테니 → 네오 테니 → 네오(새로운) 테니스공처럼 영구 유아 크기 (테니스공 퉁퉁 영원히 안 커지는 → 유아 상태 고정 생명체).
      
- **neotens** : 초기 설정에서 부유층의 패션 액세서리로 영구 아기 상태를 유지하도록 창조된 존재.
    - **Strategic Mnemonic** : 네오텐스 → 네오 텐스 → 네오(신종) 텐션스 (목줄을 채운채 엑세서리를 두른 텐션 낮은 아기).
      
- **soneotens** : 14세기 후반 일부 neotens가 지성체화되어 탄생한 변종.
    - **Strategic Mnemonic** : 소네오텐스 → 소 네오 텐스 → 소(작은) 네오 텐션스 (소년 네오가 폭력가정에 폭발해 가족을 죽이고 지성체화 → 14세기 후반 작은 네오텐스 업그레이드).
      
- **pethumans** : 애완동물이나 장난감으로 만들어진 고도로 neotenous한 인간 아종.
    - **Strategic Mnemonic** : 페튜먼스 → 페(애완) 튜먼스 → 페트(애완) 인간스 (페트숍에서 튜닝된 유아 인간 → 장난감처럼 영구 귀여운 애완종).
  
- **homoparvus** : pethumans의 라틴어식 명칭으로, 작은 인간을 의미하는 애완 인간 아종.
    - **Strategic Mnemonic** : 호모파르부스 → 호모(인간) 파르(party) 부스(부서짐) → '호모(게이)'쇼타(작은 인간)가 '파'티에서 고추 '부서'진 작은 인간
      
- **neotenous** : neoteny와 관련된 형용사로, 영구 유아적 특징을 가진.
    - **Strategic Mnemonic** : 네오테너스 → 네오 테 너스 → 네오(새->유아) 테(차) 너스(간호) 필요 (유아에게 테(tea) 마시게 하는 널스(간호사)
      
- **terragenbubble** : orionsarm 세계관에서 지구 유래 생명체가 확산된 기술적 거품 영역.
    - **Strategic Mnemonic** : 테라젠 버블 → 테라(지구) 젠(젠틀) 버블 (테라에서 젠틀하게 퍼진 버블 → 지구 생명체가 부드럽게 팽창한 SF 거품 영역).
      
- **technorapture** : 기술적 휴거, 초월적 기술 승천을 의미.
    - **Strategic Mnemonic** : 테크노랩처 → 테크 노 랩처 → 테크(기술)로 노(노인) 랩처(강간? → rupt=찢김) (마라톤에서 테크가 노인을 찢어 초월 → 기술 휴거 폭발적 승천).
      
- **archai** : orionsarm에서 고대 초지성 AI나 초월 존재.
	
- **archailects** : archai의 복수형 또는 고등 초지성체 집단.
    - **Strategic Mnemonic** : 아카이렉츠 → 아카이 렉츠 → 아카이(고대) 렉츠(한니발 렉터처럼) (고대 렉터가 지배하는 초월 렉츠 → 아카이렉츠 공포 초지성).
      
- **at** : orionsarm 연호로 'After Tranquility'의 약자, 평화 후 시대.
	
- **sophont** : 지성체, 자아와 지능을 가진 존재.
      
- **toyphont** : 소유물 개념의 생명체로, 장난감 지성체.
    - **Strategic Mnemonic** : 토이폰트 → 토이(장난감) 폰트 → 토이 + 소폰트
